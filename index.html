Competição de Dados - Construção de um modelo para prever o preço de vendas de produtos novos.

Neste projeto, utilizei os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócios, juntamente com ferramentas de desenvolvimento de modelos de machine learning e o GitHub. O objetivo era desenvolver um modelo de previsão de preços para produtos a serem lançados em uma empresa de e-commerce de roupas..

O resultado final do projeto foi um modelo regressor que prevê os preços dos produtos a serem lançados. O algoritmo escolhido foi o Random Forest, e a métrica de avaliação foi o SMAPE, alcançando uma precisão de 9,11%. Este desempenho colocou o modelo em primeiro lugar em uma competição de dados, destacando sua eficácia na previsão de preços.

Esta experiência me permitiu aprimorar minhas habilidades em programação, manipulação de dados e machine learning, demonstrando minha capacidade de aplicar esses conhecimentos em projetos práticos e desafiadores.

As ferramentas utilizadas foram:

  • Python, Pandas, Matplotlib, Numpy e Seaborn, Sklearn.
  • Visual Studio Code.
  • KMeans, Average Model, Random Forest Regressor, Lightgbm, Catboost, VotingRegressor, LinearRegressor e XGBoost Regressor.
