Construção de um modelo para previsão de vendas

Usei nesse projeto, os conceitos de Programação em Python, Estatística e técnicas de Machine Learning para fazer um modelo de previsão de vendas. E o resultado desta solução, caso fosse implementada, seria proporcionar para o CFO da empresa Rossmann, condições de consultar a previsão das vendas nas próximas seis semanas de cada uma das lojas farmacêuticas.

O resultado final do projeto foi um bot do telegram para consulta em um ambiente Cloud. O painel pode ser acessado pelo Telegram e retorna a previsão de vendas das proximas 6 semanas da loja consultada.

As ferramentas utilizadas foram:

  • Python, Pandas, Matplotlib, Numpy e Seaborn.
  • Jupyter Notebook e Visual Studio Code.
  • Average Model, Random Forest Regressor, Linear Regression Model, Linear Regression Regularized Model - Lasso e XGBoost Regressor.
  • Flask framework web, Render Cloud
  • Git e Github.
